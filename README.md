# Sistem Kalkulator Kelompok 2

## Deskripsi
Sistem Kalkulator adalah aplikasi berbasis web yang dibuat untuk memenuhi **Tugas Besar Kelompok**
pada mata kuliah **Dasar Algoritma dan Pemrograman**.  
Aplikasi ini berfungsi untuk melakukan perhitungan aritmatika dasar dan dijalankan secara lokal
menggunakan **Flask** sebagai web framework.
